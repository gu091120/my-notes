# css 上下文规则

当元素在页面上水平和垂直排布，它们之间相互作用的规则

-   格式化上下文（formatting context）:默认的文档流规则
-   行内格式化上下文（inline formatting context）:内联元素的规则
-   块级格式化上下文（block formatting context）:也就是所谓的 BFC
-   层级上下文（stacking context）:`z`轴上面的层叠关系
